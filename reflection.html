Reflection

What I learned from building this final website.

What went well?

I was able to organize the site into five complete pages and keep the design consistent. The navigation, media, table, contact form, and layout all work together as one professional website.

What was challenging?

The most difficult part was making sure every page stayed responsive and that the layout still looked clean on smaller screens. Testing the website at different sizes helped improve the final result.

What did I learn?

I learned how important planning, clean code, validation, accessibility, and consistent design are when creating a real website. This project helped me understand how HTML and CSS work together to create a professional user experience.

Final Takeaway

This portfolio represents my growth as a web developer and IT student. I am proud of the progress I made and can use these skills in future academic and professional projects.
